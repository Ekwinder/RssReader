The program takes in a RSS feed file and displays it's contents in a readable and understandable form.

The entry point to the application is Webapp.py

To start the application on your local server,
- Open a terminal
- python Webapp.py
- the server would start at  - http://0.0.0.0:8080/

Urls tested = 
http://feeds.bbci.co.uk/news/rss.xml
http://www.telegraph.co.uk/news/uknews/rss
http://rss.nytimes.com/services/xml/rss/nyt/HomePage.xml
http://techrepublic.com.feedsportal.com/c/35463/f/670841/index.rss
http://feeds.businesswire.com/BW/Automotive_News-rss
